## 神经网络发展历程
```
1、MCP：输入加权+求和+激活函数阈值过滤输出。只能模拟一个神经元
2、感知器：梯度下降算法实现自学习，多维输出分类。只能处理线性问题，不能异或（既非又非）分类
3、多层感知器：BP反向传播算法+非线性激活函数，解决非线性问题。基于单个感知器构建多层网络，解决复杂问题。
4、卷积：拟合一切。LeNet
5、ReLU+AlexNet：抑制和解决梯度消失问题，使用BP算法在层数较多时误差传递到前层会衰减导致无法训练。
```
